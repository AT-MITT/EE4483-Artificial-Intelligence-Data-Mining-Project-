# EE4483-Artificial-Intelligence-Data-Mining-Project-
Project 2: Cats vs Dogs Classification

# Steps to run
This code has been tested on Python 3.10.4.

Ensure Pytorch, Torchvision, numpy, pandas, matplotlib, scikit-learn, tqdm, pillow, jupyter, and seaborn are installed

Extract the dataset

Adjust all directory addresses according to your file locations

Run the training py file for the intended model and dataset to obtain the best fit weights pth file and plots of training loss, validation loss, and learning rate, for example Standard_Inception for training the Inception V3 model on the Cats vs Dogs dataset

Run the respective testing py file to obtain the evaluation results
